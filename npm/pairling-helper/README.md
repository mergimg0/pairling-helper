# pairling-helper

This name is reserved by the Pairling team to prevent confusion with the
historical "Pairling Helper" Mac artifact name.

**The real package is [`pairling`](https://www.npmjs.com/package/pairling):**

```sh
npm install -g pairling
pairling setup
```

- Source mirror: https://pairling.dev
