# pairlingd

This name is reserved by the Pairling team (it matches the Pairling Mac
daemon's internal name) to prevent confusion and typosquatting.

**The real package is [`pairling`](https://www.npmjs.com/package/pairling):**

```sh
npm install -g pairling
pairling setup
```
